# PDF RAG Chatbot

A local RAG-based PDF chatbot built with Streamlit, LangChain, Ollama, and ChromaDB.

## Features
- Upload any PDF and chat with it
- Conversation memory
- Streaming responses
- Source citations
- Clear chat / upload new PDF

## Requirements
- Python 3.10+
- Ollama running locally with `llama3.2:1b` and `nomic-embed-text` pulled

## Installation
pip install -r requirements.txt

## Run
streamlit run app.py
